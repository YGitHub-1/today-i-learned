# Array(배열)

<br>

## 배열이란?

 : 번호(인덱스)와 번호에 대응하는 데이터들로 이루어진 자료 구조  
 
<br>
 
## 배열의 장점

 : 연관된 데이터를 저장하기 위한 변수의 선언을 줄여주며, 반복문 등을 이용하여 계산과 같은 과정을 쉽게 처리할 수 있음  

<br>

## 배열 선언

- 1차원 배열 선언
    
    `int [] arr;`
    
    `int arr[];`

- 1차원 배열 초기화

    `int[] arr = new int[size];`

    `int[] arr = {1, 3, 5, 7, 9};`

- 2차원 배열 선언
    
    `int [][] arr;`
    
    `int arr[][];`
    
    `int [] arr[];`

<br>

## 배열 탐색

```java
for (int i = 0; i < 배열명.length; i++) {
    실행문;  // 배열명[i]
}
```

```java
for (자료형 변수명: 배열명) {
    실행문;
}
```
