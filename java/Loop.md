## 반복문

- 조건이 true인 동안 반복 수행
- for 문 (jdk 1.4형식)
    
    ```java
    for (초기값; 조건값; 증감값) {
    	수행문;
    }
    ```
    
    초기값 : 0 ~ N개, 같은 타입
    
    조건식 : 0, 1개
    
    증감값: 0 ~ N개
    
    조건식 횟수만큼 반복 수행
    
- for-each 문 (jdk 1.5 형식 : 배열, collection)
    
    ```java
    for (타입 변수명: 배열명 또는 collection 참조 변수명) {
    	수행문;
    }
    ```
    
    배열 또는 collection에 저장된 갯수만큼 순서대로 하나씩 가져와서 변수에 할당
